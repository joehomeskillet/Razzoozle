Expand description

A trait used to extract the arguments from the disconnect event. The Result associated type is used to return an error if the extraction fails, in this case the DisconnectHandler is not called.

  • See the disconnect module doc for more details on disconnect handler.
  • See the extract module doc for more details on available extractors.
