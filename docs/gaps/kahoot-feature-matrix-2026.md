# Kahoot Feature Matrix — Stand 2026

## Fragetypen

| Feature | Beschreibung |
|---------|-------------|
| **Multiple Choice** | Klassische 4-Option Single-Select, optionale Multi-Select-Variante |
| **True/False** | Binäre Ja/Nein-Fragen mit Direktfeedback |
| **Type Answer** | Freieingabe mit automatischer Antwort-Validierung (exact/fuzzy matching) |
| **Puzzle / Sequencing** | Reihenfolgefragen — Elemente sortieren und anordnen |
| **Slider / Rating Scale** | Numerische Skalenfragen (1-10, Prozent, custom range) |
| **Poll / Survey** | Unkorrektor Umfrage, keine Punkte, Visualisierung der Antwortverteilung |
| **Word Cloud** | Freieingabe mit Visualisierung häufigster Antworten |
| **Brainstorm** | Kollaborative Ideensammlung, alle Antworten sichtbar, moderierbar |
| **Drop Pin / Image Hotspot** | Bild-basierte Fragen — Punkt auf Bild klicken oder Gebiet eingrenzen |
| **Confidence Rating** | Kombination aus Antwort + Selbsteinschätzung (sicher/unsicher) |
| **Matching** | Paare zuordnen (z.B. Begriff ↔ Definition) |
| **Fill in the Blank** | Lückentexte mit Multi-Answer-Support |
| **Diagram Labeling** | Beschriftung von Diagrammen/anatomischen Bildern |
| **Vocabulary Review** | Strukturierte Vokabel-Sessions mit Aussprache |
| **Micro-Lessons** | Kurzvideo + nachfolgende Verständnisfrage |
| **Practice Tests** | Mehrteilige Übungsszenarien mit Feedback pro Frage |

## Game-Modi

| Feature | Beschreibung |
|---------|-------------|
| **Classic Live** | Synchrone Echtzeit-Session mit Host und Spielern, Leaderboard |
| **Team Mode** | Gruppenwettbewerb, pro Team ein Gerät, höhere Punktwerte (Premium) |
| **Self-Paced / Assignments** | Asynchrone Hausaufgaben, Spieler folgen eigenem Tempo, Deadline-Unterstützung |
| **Study Mode / Flashcards** | Individuelle Lernmodus mit Wiederholungslogik (Spaced Repetition optional) |
| **Practice Mode** | Unbewertete Trainingsübungen mit detailliertem Feedback |
| **Ghost Mode / Replay** | Virtueller Gegner (AI/früherer Top-Spieler), offline Spielbar |
| **Challenge Mode** | Asynchrone Herausforderungen an Freunde/Klassenkameraden |
| **Kahoot! Jumble** | Wort-Ratespiel, 2–4 Wörter aus Buchstaben bilden |

## Host-Features

| Feature | Beschreibung |
|---------|-------------|
| **Live Session Management** | Host-Dashboard, Spieler-Join-Monitor, Pause/Resume, Spieler-Kick |
| **Live Q&A Panel** | Echtzeit-Fragen von Spielern, Host kann Upvotes sehen und beantworten |
| **End-of-Game Summary** | Automatische Zusammenfassung mit Top 3, Best Question, Most Answered |
| **One-Click Archive** | Sessions für spätere Analyse speichern, Replay starten |
| **Lobby Music & Themes** | 50+ vordefinierte Musikstücke, einfache Theme-Auswahl (Dunkel/Hell) |
| **Immersive Branding** | 112+ Schriftarten, Custom Logo Upload, Brand-Farben, Hintergrundbild |
| **Podium / Leaderboard** | Dynamisches Ranking, Position-Animationen, Tie-Handling |
| **Streaks & Bonus Points** | Bonus für richtige Antworten in Folge, Momentum-Multiplier |
| **Player Identifier** | Name, Avatar, Emojis, Custom Ident pro Session wählbar |
| **Randomize Questions** | Fragen oder Antwortoptionen zufällig mischen |
| **Participant Cap Control** | Host kann Max. Spieler limitieren, spätere Zutritte ablehnen |
| **Session Replay** | Aufzeichnung abspielen, Statistiken wiedersehen |
| **Live Stats Dashboard** | Detaillierte Chart-Ansicht pro Frage, Antwortverteilung in Echtzeit |
| **Screenshot/Export Results** | Ergebnisse als Bild/PDF exportieren oder teilen |
| **Timer Control** | Frage-Anzeigedauer anpassen, Countdown-Optionen |
| **Bulk Question Import** | CSV/Excel Import für schnelle Erstellung mehrerer Fragen |
